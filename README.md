# CucurbitGenomics
Pipelines and scripts for Cucurbitaceae genomics and evolution
Bioinformatics scripts and pipelines for Cucurbitaceae (cucumber, bottle gourd, watermelon, squash) population genomics, pangenome construction, SV analysis, and evolutionary biology.

## Contents
- **Genome assembly and annotation**
- **Pangenome graph construction**
- **Structural variant discovery and genotyping**
- **Population genetics**
- **GWAS and genomic prediction**
- **Adaptation**

## Species included
- *Cucumis sativus* (cucumber)
- *Lagenaria siceraria* (Molina) Standl. (bottle gourd)
- *Citrullus lanatus* (watermelon)
- *Cucurbita pepo* (squash/pumpkin)
